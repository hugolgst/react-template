## How to use
### Using NodeJS
Install of the neccesary packages:
```bash
$ npm i
```

#### Test
```bash
$ npm run dev
```

#### Production
```bash
$ npm run build
$ npm run start
```

### Docker
```bash
$ docker build . -t react-template
$ docker run -p 3000:3000 react-template
```
